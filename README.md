# TarantoCaffè - Mattia Melucci

Progetto di una landing page moderna e interattiva per una caffetteria d'élite, focalizzata su design pulito ed esperienza utente fluida.

## 🛠️ Tecnologie Utilizzate
- **HTML5**: Struttura semantica.
- **Tailwind CSS**: Styling avanzato e responsive design.
- **JavaScript (Vanilla)**: Gestione dello slider hero e interazioni dinamiche.
- **Lucide Icons**: Iconografia vettoriale.
- **AOS (Animate On Scroll)**: Animazioni all'entrata dei contenuti.

## 📋 Caratteristiche principali
- **Manutenibilità**: Codice organizzato in file HTML, CSS e JS distinti.
- **Full Responsive**: Ottimizzato per ogni dispositivo.
